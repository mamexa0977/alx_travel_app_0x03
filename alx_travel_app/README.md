# ALX Travel App - Payment Integration with Chapa API

## Overview
This project integrates the Chapa Payment Gateway into a Django-based travel booking application. It provides secure payment initiation, verification, and status handling for bookings.

## Features
- Secure payment processing with Chapa API
- Payment status tracking and management
- Automatic confirmation emails
- Webhook handling for payment notifications
- Background task processing with Celery
- RESTful API endpoints

## Prerequisite s 
- Python 3.8+
- PostgreSQL
- Redis (for Celery)
- Chapa API account (sandbox credentials)
